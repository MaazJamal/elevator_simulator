This function removes adds a passenger from a queue to the elevator_array

This function take in an elevator, index of the elevator, array of floor and current_floor. and removes the passenger from the floor and adds it to elevator passenger array.

Parameters
[in]indexint value that specifies the index of the elevator. index start at 0.
[in,out]elevator_arrpointer to elevator array.
[in,out]floor_arrarray of pointers to passengers on floor.
[in]cur_florrint value that specifies the current floor. As described in elevator so start at 1 and equal to max_floor_count.
[in]event_timeunsigned int the time the event has happened.
Returns
return the number of passengers added to the elevator.
