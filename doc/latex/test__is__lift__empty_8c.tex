\begin{DoxyAuthor}{Author}

@detail The function tests with different values if the output comes correct or not. elevator\+\_\+count which is a global variable has been set to a fix value The test cases are as below\+:
\begin{DoxyItemize}
\item For array with passenger count equal to the maximum number of passengers of the lift, and index value as 0 or less than the elevator count the output is 0
\item For array with passenger count less than the maximum number of passengers of the lift, and index value as 1 or less than the elevator count the output is 1
\item For array with passenger count more than the maximum number of passengers of the lift, and index value as 2 or less than the elevator count the output is 0
\item For array with index value as 3 or greater than or equal to the elevator count the output is -\/1
\item For an empty array, the output is -\/1 and the program exits
\end{DoxyItemize}
